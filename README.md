
# MNC-Frontend

This project is a full managment system for Minecraft networks.

## Installation

Install MNC-Frontend with npm

```bash
  git clone https://github.com/MinecraftNC/frontend.git
  cd frontend
  yarn install
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/MinecraftNC/frontend.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run start
```
